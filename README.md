## What Is Alpha Notes?

Alpha Notes is a lightweight, static research blog built using **pure HTML, CSS, and JavaScript**, designed to present finance research ideas clearly and intuitively.

The blog focuses on:
- Empirical market anomalies  
- Behavioral finance insights  
- Quantitative investment strategies  
- Risk, volatility, and return dynamics  

Each post distills academic research into accessible explanations while preserving theoretical depth.
